## Run Locally

Clone the project

```bash
  git clone https://github.com/omkundu/affable-fang-3484
```

Go to the project directory

```bash
  cd mytheresa
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
